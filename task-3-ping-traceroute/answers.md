# Task 3 — Ping and Traceroute Experiments

## Commands Used

```bash
ping 192.168.0.1
ping 8.8.8.8
ping 1.1.1.1
tracert google.com
```

---

## Питання та відповіді

### Скільки хопів (проміжних маршрутизаторів) до google.com?

Всього хопів 30 але відповів тільки один це роутер:

```text
1    <1 ms    <1 ms    <1 ms    192.168.0.1
```

Інші мовчать через firewall.

---

### Який середній час відгуку (RTT - Round Trip Time)?

Середній час відгуку = 0ms

```text
Approximate round trip times in milli-seconds:
Minimum = 0ms, Maximum = 0ms, Average = 0ms
```

---

### Чи всі проміжні маршрутизатори відповідають? Чому деякі можуть не відповідати?

Ні, відповідає тільки роутер.

Інші маршрутизатори можуть не відповідати через firewall або блокування ICMP-пакетів.

---

### Яка різниця між ping до локального шлюзу та публічного сервера?

Локальний шлюз `192.168.0.1` — відповідає, RTT = 0ms, бо він поруч.

Публічний сервер `8.8.8.8` — не відповідає, бо знаходиться далеко + firewall блокує ping.
