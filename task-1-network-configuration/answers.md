# Task 1 — Network Configuration

## Commands Used

```bash
ipconfig /all
route print
arp -a
```

---

## 1. Network Interfaces

У моїй системі 4 мережевих інтерфейсів.

### Їхні MAC-адреси:

- Wireless LAN adapter Підключення через локальну мережу* 1  
  MAC: E8-C8-29-CD-25-6D

- Wireless LAN adapter Підключення через локальну мережу* 2  
  MAC: EA-C8-29-CD-25-6C

- Wireless LAN adapter Wi-Fi  
  MAC: E8-C8-29-CD-25-6C

- Ethernet adapter Ethernet  
  MAC: 2C-58-B9-DA-0D-02

---

## 2. IP Address and Subnet

Моя Ip-адреса: `192.168.0.6`

Я належу до підмережi: `192.168.0.0`

---

## 3. Default Gateway

Адреса шлюзу за замовчуванням: `192.168.0.1`

Це мій роутер, який допомагає вийти назовні в інтернет.

---

## 4. ARP Cache

У моєму ARP-кеші 10 записів.

1 реальний пристрій — роутер `192.168.0.1`, його MAC-адреса збережена щоб не запитувати кожного разу.
