# Network Protocols Research
This project contains practical research and experiments with network protocols (IP, Ethernet, MAC, ARP, ICMP) and subnetting, including C# implementations for network analysis tools.

## Tasks

The project is divided into the following practical labs:

- Task 1 — Network configuration analysis (ipconfig, route, arp)
- Task 2 — Subnet calculations
- Task 3 — Ping and traceroute experiments
- Task 4 — ARP protocol investigation
- Task 5 — C# IP Address Parser
- Task 6 — Network Scanner (C#)
- Task 7 — Ethernet frame analysis (Wireshark)
- Task 8 — OSI model diagram
- Task 9 — Loopback interface study
